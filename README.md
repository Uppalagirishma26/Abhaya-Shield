# Abhaya-Shield
Abhaya Shield is an AI-inspired safety and driver verification app that provides complete driver details in a single click. Users can verify a driver using a photo or manual name entry to instantly view traffic violations, police records, and a trust score, helping improve women and passenger safety during rides.
## Features
- One-click driver verification using photo
- Manual name-based driver search
- Traffic violation details
- Police record simulation
- Trustworthiness score
- Consent-first safety design

## Tech Stack
- React (TypeScript)
- Tailwind CSS
- Lucide Icons

## Project Flow
Consent → Upload (Photo / Name) → Processing → Result

## Future Enhancements
- Offline alert messaging to emergency contacts
- Voice-activated SOS alerts
- Automatic danger detection
- Route misdirection detection

## Disclaimer
This is an educational prototype. No real police database or face recognition is used.
